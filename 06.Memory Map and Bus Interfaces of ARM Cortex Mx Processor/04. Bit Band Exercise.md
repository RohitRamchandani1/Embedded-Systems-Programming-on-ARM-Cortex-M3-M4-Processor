# Bit Banding Exercise
## Exercise
- Modify the content of the memory location 0x2000_0200 using usual and bit banding method and analyze the difference.

- First store the value of the 0xff into the memory location 0x2000_0200.

- Make the 7th-bit position of the value to 0.

## Calculation of bid band alias address
- Calculate the `bit band alias address` for given `band memory address`.

- 7th bit position of the memory location 0x2000_0200 using its alias address.

- General Formula for Alias Address Calculation: 

- `Alias address = alias_base + (32 * (bit_band_memory_addr - bit_band_base)) + bit * 4`

- Alias Base Address can be fetched from the Memory Map = 0x2200_0000 

- Alias Address = 0x2200_0000 + (32 * (0x2000_0200 - 0x2000_0000)) + 7 * 4


```c
#include <stdint.h>
#define ALIAS_BASE 0x22000000
#define BITBAND_BASE 0x20000000
int main(void)
{
    /* Pointer variable to point to the address */
    uint8_t *ptr = (uint8_t*)0x20000200;

    /* Reset to 0xff */
    *ptr = 0xff;

    /* Clearing the 7th bit position. */
    *ptr &= ~(1 << 7);

    /* Reset to 0xff */
    *ptr = 0xff;

    /* Bit Band Method */
    uint8_t *alias_base = (uint8_t *)(ALIAS_BASE + (32 * (0x20000200 - BITBAND_BASE)) + 7 * 4);

    /* Clearing the 7th bit of the address 0x20000200 */
    *alias_addr = 0;

    for(;;);
}

```

- This feature is available for the SRAM region and the peripheral region.
