## Example 2: Move the content of the CONTROL register to 'C' variable "control_reg"
- CONTROL register is a special register of the ARM core.
- To read CONTROL register you have to use MRS instruction.

`Instruction` ==> MRS
`Source`      ==> CONTROL register
`Destination` ==> A 'C' variable "control_reg" (OUTPUT operand)

```c
    uint32_t control_reg;
    __asm volatile("MRS %0, CONTROL":"=r"(control_reg)::);
```

- The MSR and MRS instruction are used to transfer data from and to the special purpose registers.

- `MSR` is used to move the contents of a special register to a general purpose register.
- Syntax: `MRS{cond} Rd, spec_reg`

- `MRS` is used to move the contents of a general purpose register to a special register.
- Syntax: `MSR{cond} spec_reg, Rd`


```c
int main()
{
    int control_reg;

    /* Read CONTROL register to control_reg */
    __asm volatile("MRS %0, CONTROL":"=r"(control_reg)::);

    for(;;);
}
```

## Example 3: Move the content of 'C' variable var1 to var2 using inline assembely coding
`Instruction` ==> MOV
`Source`      ==> a 'C' variable 'var1' (INPUT operand)
`Destination` ==> a 'C' variable 'var2' (OUTPUT operand)

```c
int main()
{
    int var1=10, var2;
    __asm("MOV %0,%1": "=r"(var2): "r"(var1));
}
```

## Example 4: Copy the contents of a pointer into another variable

```c
int main()
{
    int p1, *p2;
    p2 = (int *)0x20000008;
    __asm volatile("LDR %0, [%1]":"=r"(p1):"r"(p2)); /* p1 = *p2 */
}
```
